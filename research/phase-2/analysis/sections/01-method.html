Method

Phase 1 had two caps. Programs were contrived – no tools, no judgment, no world knowledge – and traces were self-reported by the model, which made a formatting lapse indistinguishable from an execution error. Phase 2 removes both. Programs take real procedure shapes (runbooks, checklists, policies) with deterministic invariants threaded through the control flow, and the harness observes the trace through three channels, none of them self-reported:

Session transcripts Agentic runs go through claude -p; the full session transcript is read back from disk. Task tool calls are SPAWN events, carrying the subagent type and the complete WITH payload.
MCP tool log A mock state server (get/set) appends every call to a log file. The set log doubles as an assignment trace, captured mechanically.
Canary tokens Hybrid programs thread a unique token through each item of the data flow. Token presence in worker prompts and final output makes fan-out coverage and aggregation deterministic to score, with no judge.

Where a run is single-turn and the final output is the thing measured – E1's halt behaviour, E4's decision line – phase-1-style output scoring is kept, because no intermediate trace is being trusted. The headline metric where traces are compared is unchanged from phase 1: step accuracy, the fraction of the reference trace the model reproduced in order, computed as the longest common subsequence of the two traces. The reference comes from a deterministic interpreter, so each program has exactly one correct execution.

Fuzzy leaves – the prose work inside a procedure – are scored by an LLM judge (Claude Sonnet, 40 calls, order counterbalanced). The judge sees the procedure and the output together and scores recognisable compliance from 0 to 2. Judges are reliable at recognising compliance when shown both, even where they would not have produced the output themselves. Cheap models remain the case of interest – Claude Haiku and GPT‑5.4 mini carry most cells, with Sonnet at anchor cells – because a procedure that only runs on a frontier model is doing little work.
