Results against the research brief

QuestionPhase-2 result
Q1 — can notation enforce integrity (halt, not repair)? Split by fault class. The pragma flipped wrong type annotations from silent coercion to a 10/10 halt, with one clean run in ten wrongly halted on one model. Syntax faults halted 1/10 (§1).
Q2 — where does size break execution? No breaking point appeared within the tested range. The phase-1 slip at 200 statements did not reproduce, and every observed failure occurred on programs well inside the executable range (§5).
Q3 — do procedures steer, or relabel? The procedure arm passed the checker 18/20 (20/20 on decisions) against 7/20 for the goal arm, with the gap concentrated on tasks where policy contradicts the helpful default; judge adherence 1.95 vs 0.6 (§4).
Q4 — do parameters bind across skill/agent boundaries? Payload and lambda fidelity were 1.0 on both models, through a higher-order skill into real subagent fan-out. The observed failures were tool routing and worker refusal (§2).
Q5 — does external state extend executable size? Not answerable with this design: models did not maintain instructed journalling, so a voluntary store neither extended size nor provided the audit trail (§3).
