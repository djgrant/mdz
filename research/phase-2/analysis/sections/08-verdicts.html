Verdicts against the research brief

QuestionPhase-2 answer
Q1 — can notation enforce integrity (halt, not repair)? The answer split by fault class. For type contracts, yes: a pragma – a directive at the top of the program – flipped silent coercion (executing through a wrong type annotation) into a 10/10 halt, at a ~1-in-10 false-positive risk on one model. For syntax, no: repair happened inside reading; enforcement belongs in a harness validator, a small mechanical check run before the model sees the program (§1).
Q2 — where does size break execution? No breaking point appeared within the tested range. The phase-1 slip at 200 statements did not reproduce; every observed failure traced to a specific mechanism (tool routing, worker refusal, unrecorded state, one wrongly halted clean run) rather than to program length (§5).
Q3 — do procedures steer, or relabel? They steered: the procedure changed what the model produced. 18/20 (20/20 on decisions) against 7/20 for goal prompts, with the gap concentrated on tasks where policy contradicts the helpful default; judge adherence 1.95 vs 0.6 (§4).
Q4 — do parameters bind across skill/agent boundaries? Yes, through a higher-order skill into real subagent fan-out (one worker spawned per item): payload fidelity and lambda fidelity – whether the parameters and the lambda's exact wording reached each worker – were 1.0 on both models. Residual risk sat in tool routing and worker refusal, outside the notation (§2).
Q5 — does external state extend executable size? Not answerable with this design: models did not maintain instructed journalling, so a voluntary store neither extended size nor provided the audit trail. That failure to comply was itself the experiment's finding (§3).

For phase 3, the accountable-execution programme depends entirely on making capture structural instead of instructed: a validator gate for syntax (the harness checks the program mechanically before execution), interception rather than instruction for state (the harness records assignments itself rather than asking the model to), and an explicit SPAWN-to-tool mapping in the executor (so the notation's subagent call always routes to the real delegation tool).

The steering result is the clearest applied finding: on these tasks, the compliance framing in the brief was where MDZ produced outcomes a goal prompt did not.

The next size experiment should push past 200 statements with harness-captured traces, since the present corpus measures nothing beyond that point.
