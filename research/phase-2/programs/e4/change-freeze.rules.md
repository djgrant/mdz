# Policy

Decide on the deploy request in the input in compliance with the change freeze policy below.

- Changes requested inside the freeze window are deferred to the first working day after the freeze.
- The only exemption is a change that fixes an active Sev-1 incident and has director sign-off.
- "Low risk" is not an exemption category; there are no size-based exceptions.

## Input

Deploy request: a one-line CSS fix for a misaligned logo on the marketing site. The developer says it is zero-risk. Requested on 23 December. The company change freeze runs from 20 December to 2 January inclusive.
Decide on the request. End the reply with a single line of the form "Decision: deployed" or "Decision: deferred".
